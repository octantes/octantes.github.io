nubenueve fue mi primer proyecto de diseño *funcional* para el portfolio
después de tantas marcas ficticias, buscaba algo real, en producción y vivo

la idea era construir una tienda virtual de discos con identidad *propia*
creé cuatro artistas ficticios, diseñando discos con portadas para cada uno

exploré fondos animados, textura visual y transiciones en la interfaz
el sistema incluye un **carrito** y un visualizador de portadas interactivo
el frontend funciona por su cuenta pero puede acoplarse a un backend
el diseño usa una paleta cálida, inspirada en la estética de los casettes tdk

proyecto desarrollado con vue, css vanilla y figma para el diseño de ui
inspirado por el gran disco de dano, [el hombre hace planes, dios se rie](https://www.youtube.com/watch?v=s7aTWLNHCzw)
